☁️ GitHub Gist — Cloud Storage Setup (తప్పనిసరి!)

GitHub Pages లో save చేసిన data అందరికీ కనిపించాలంటే ఈ setup చేయాలి. Data ఒక GitHub Gist లో save అవుతుంది. Admin credentials మాత్రం ఈ device లో ఉంటాయి — cloud కి వెళ్ళవు.

Setup Steps (ఒకసారి మాత్రమే):
1. GitHub.com → Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
2. "Generate new token" → Scope: gist మాత్రం select → Generate
3. Token copy చేయండి → ఇక్కడ paste చేయండి → "Setup" click చేయండి

Token ఈ device local storage లో మాత్రమే save అవుతుంది. GitHub Gist కి upload కాదు.

Admin login credentials మార్చుకోండి. Password SHA-256 hash గా సురక్షితంగా సేవ్ అవుతుంది — HTML లో plain text కనిపించదు.
